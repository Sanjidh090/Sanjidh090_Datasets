# Contributing Guidelines

Thanks for your interest in improving the BUETIAN Rhapsody dataset!

## Ways to Contribute
- Report issues (bad timestamps, duplicates, broken URLs).
- Suggest additional metadata fields (e.g., post category).
- Propose preprocessing scripts or labeling guidelines.
- Share reproducible notebooks (EDA, models, fairness analyses).

## Submitting Changes
1. Open a Kaggle Discussion post with:
   - A clear summary of the issue or proposal.
   - Evidence (row indices, URLs, screenshots).
   - Suggested fix or PR link (if hosted on GitHub).
2. The maintainer will review and include in the next versioned release.

## Code of Conduct
Be respectful. No harassment, deanonymization attempts, or targeted attacks.
